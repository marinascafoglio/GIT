How to Integrate Gulp and Browser Sync for Auto Reloading HTML Pages

1

Para instalar Gulp hay que tener previamente instalado node.js, se descarga la version desde la web oficial y nada m치s que hacer en este primer paso.

2

En el terminal. Una vez estamos dentro del directorio de nuestro proyecto, por ejemplo:

cd ~/Documents/BOTTEGA/02_FULLSTACK/04_GIT

tecleamos "sudo npm install -g gulp". Es necesario teclear la palabra sudo en Mac, para poder tener los permisos de administrador, sino no se instala correctamente. Nos pide la contrase침a, tecleamos (aunque no aparece escrita) y pulsamos enter. Ya est치 instalado.

3

Si ahora vamos a nuestro VSCode, vemos que se han creado algunos archivos en la carpeta de nuestro proyecto.
Para poder continuar sin todas las actualizaciones de git (miles), debemos crear un archivo dentro de la carpeta del proyecto llamado .gitignore y dentro escribir "node_modules" y listo.

4

Ya podemos crear nuestro archivo con la funcion de gulp. se llamara gulpfile.js y dentro escribiremos el siguiente codigo:

var gulp = require('gulp');
var browserSync = require('browser-sync').create();
var reload = browserSync.reload;

gulp.task('serve', function() {
browserSync.init({
server: {
baseDir: "./"
}
});

gulp.watch("*.html").on("change", reload);
});

5

Para ejecutar nuestro procesador de tareas (Gulp) en el terminal, escribimos "gulp serve" y se abre la pagina en el navegador, los cambios que realicemos en nuestro proyecto los veremos en directo. Para poder seguir tecleando en el terminal una vez que ejectuamos "gulp serve" debemos teclear los comandos control+c de esta forma ya tenemos operativo el terminal.

6

Para actulizar nuestro repositorio GIT, en el terminal escribimos "git add ." luego: git commit -m "Installed gulp for task running", ya tenemos los archivos listos para ser enviados al repositorio, ahora solo falta escribir "git push" y hecho!

*

Cada vez que salgamos del terminal o de nuestro proyecto, o cerremos el navegador, la conexi칩n con el navegador se pierde por lo que tenemos que repetir el comando "gulp serve". O si hemos cerrado el terminal, debemos primero llegar al directorio y luego ejecutar Gulp.
