tercero: Si ahora vamos a nuestro visual studio, vemos que se han creado algunos archivos en la carpeta de nuestro proyecto.

cuarto: ya podemos crear nuestro archivo con la funcion de gulp. se llamara gulpfile.js y dentro escribiremos el siguiente codigo:
var gulp = require('gulp'); var browserSync = require('browser-sync').create(); var reload = browserSync.reload; gulp.task('serve', function() { browserSync.init({ server: { baseDir: "./" } }); gulp.watch("*.html").on("change", reload); });

sexto: para ejecutar nuestro procesador de tareas (gulp) en el terminal, escribimos "gulp serve" y se abre la pagina en el navegador, los cambios que realicemos en nuestro proyecto los veremos en directo. Para poder seguir tecleando el terminal una vez que ejectuamos el navegador con gulp debemos teclear los comandos control+c de esta forma ya tenemos operativo el terminal.

ULTIMO: para actulizar nuestro repositorio GIT, en el terminal escribimos "git add ." luego "git commit -m "Installed gulp for task running", ya tenemos los archivos listos para ser enviados al repositorio, ahora solo falta escribir "git push" y hecho!
